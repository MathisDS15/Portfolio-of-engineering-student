Formation

Je suis actuellement en deuxième année d'école d'ingénieur en France. Je suis spécialisé en cloud, systèmes et sécurité cette année. Cependant, l'année dernière j'ai appris beaucoup de choses en informatique et maintenant je suis capable de créer des applications web, des systèmes embarqués, ...

Projets

Password Checker

Cette application web Flask traite le problème répandu des mots de passe faibles ou réutilisés. Elle analyse leur force en temps réel, affiche des indicateurs clairs de vulnérabilité et donne des conseils pratiques pour les renforcer. L'interface HTML/CSS épurée combinée à un backend Python rend la création de credentials sécurisés simple et éducative.

Technologies

- Framework : Flask
- Frontend : HTML & CSS
- Backend : Python

Learn & Build for Web

Learn & Build for Web est une plateforme interactive Next.js qui rend le démarrage du développement web moins intimidant. Elle transforme l'apprentissage HTML, CSS, JavaScript, PHP et MySQL en jeu grâce à des exercices progressifs, des aperçus en temps réel et une validation instantanée. Son interface moderne et responsive transforme un programme aride en une expérience pratique engageante.

Technologies

- Framework : Next.js
- Frontend : HTML, CSS et JavaScript

H2O Reminder

H2O Reminder est une application iOS React Native conçue pour aider les utilisateurs à se souvenir de boire suffisamment d'eau. Elle définit des objectifs quotidiens personnalisés, envoie des rappels locaux à des intervalles choisis et suit chaque boisson avec un tableau de bord intuitif. Développée avec Expo et AsyncStorage, elle offre un projet mobile complet testé sur iPhone via TestFlight.

Station Météo

Ce prototype de station météo basé sur STM32 mesure la température, l'humidité, la pression et la pluie. Il simule également la rétraction automatique du linge avec un servomoteur pour montrer comment le matériel connecté peut automatiser les tâches. Le code embarqué, la conception du PCB et un boîtier imprimé en 3D sont combinés dans un seul appareil fonctionnel.

Technologies

- Langage : C
- CAO : Inventor
- PCB : Altium Designer
