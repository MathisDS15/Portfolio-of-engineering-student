Password Checker

This Flask web app addresses the widespread issue of weak or reused passwords. It analyses their strength in real time, displays clear indicators of vulnerability, and gives practical tips to strengthen them. The clean HTML/CSS interface combined with a Python backend makes secure credential creation straightforward and educational.

H2O Reminder

H2O Reminder is a React Native iOS app built to help users remember to drink enough water. It sets personalised daily goals, sends local reminders at chosen intervals and tracks every drink with an intuitive dashboard. Developed with Expo and AsyncStorage, it offers a full mobile project tested on iPhone via TestFlight.

Meteo Station

This STM32-based weather station prototype measures temperature, humidity, pressure and rain. It also simulates automatic laundry retraction with a servo motor to show how connected hardware can automate tasks. Embedded code, PCB design and a 3D-printed enclosure are combined in a single functional device.
